# Sitewave

**Version 0.2**

Ein leistungsfähiger Editor für statische Webseiten, entwickelt mit Python und PyQt6.

### Features

* Markdown Editor mit Live-Vorschau
* Frontmatter Management
* Plugin System & Snippets
* Integrierter Static Site Generator
